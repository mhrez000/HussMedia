# HussMedia

Marketing website for **HussMedia** — a photography & videography studio specializing in:

- **Weddings** — photo coverage & cinematic film
- **Real Estate** — HDR listing photography, walkthrough video & aerial drone tours
- **Business** — brand photography & corporate video

## Stack

Plain static HTML / CSS / JS — no build step.

- `index.html` — the site
- `styles.css` — editorial design (ivory + charcoal + gold), fully responsive
- `script.js` — mobile nav + footer year
- `robots.txt`, `sitemap.xml` — SEO / crawling

## Run locally

Any static server works, e.g.:

```bash
python -m http.server 5599
```

Then open http://localhost:5599

## Before launch

Replace placeholders: contact details, `hussmedia.com` URLs, the Formspree form `action`, and swap the demo Unsplash images for real galleries.
